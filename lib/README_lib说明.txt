此文件夹用于存放 Tess4J 所需的所有 .jar 库文件。

请将以下文件手动下载并放入本目录中（lib/）：

- tess4j.jar
- jna.jar
- jna-platform.jar
- lept4j.jar
- slf4j-api.jar
- slf4j-simple.jar
- commons-io.jar
- commons-lang3.jar

所有开发成员必须导入这些 .jar 文件到 NetBeans 中：
→ 项目右键 > Properties > Libraries > Classpath > Add JAR/Folder
→ 选择本 lib/ 文件夹下所有 .jar 文件
